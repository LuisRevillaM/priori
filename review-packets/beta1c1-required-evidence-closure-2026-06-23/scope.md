# Scope

## In Scope

- Required evidence completeness for the attested Hermes hero.
- Public execution completeness fields.
- Cache invalidation for stale incomplete executions.
- N1D and N1D.1 proof gate hardening.
- Workbench incomplete-execution warning.
- Product copy correction for previously attested Hermes origin.
- Render readiness pin alignment with regenerated knowledge pack.

## Out Of Scope

- New Hermes prompts or hero wording.
- New tactical capabilities, primitives, or operators.
- A new live Hermes compile.
- New UI architecture or visual polish.
- Second tactical family work.
- Feedback/revision workflow.

## Assumptions

- The full repository, canonical data, and cloud credentials are not included in this packet.
- Reviewers can inspect included artifacts and route responses, but cannot independently rerun the full validation from the packet alone.
