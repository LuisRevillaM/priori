# Scope

## In Scope

- Review of the proposed M1.1 / M1.2 split.
- Review of M1.1 runtime scope, gates, non-goals, and acceptance criteria.
- Review of M1.2 Hermes/workshop scope, gates, non-goals, and acceptance criteria.
- Review of downstream implications for M2-M6.
- Review of reward-hacking risks, missing gates, and implementation order.

## Out of Scope

- Re-verifying M1 implementation artifacts.
- Running local data/code.
- Reviewing raw IDSSE data.
- Reviewing UI implementation.
- Reviewing production deployment, provider integration, or video.
- Approving any code implementation for M1.1/M1.2.

## Packet Type

`inspection_packet_only`

The external reviewer can inspect the plan and provide architecture feedback. They cannot reproduce local verification without repo access and local data artifacts.
