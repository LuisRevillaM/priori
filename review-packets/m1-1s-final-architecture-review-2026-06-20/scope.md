# Scope

## Included

- M1.1S final local architecture proof after S4R, S5, S6, and S7.
- Runtime source files needed to inspect generic execution, evidence projection, and compatibility split.
- Three query plans:
  - approved M1 legacy parity plan;
  - S4 opposite corridor after shift plan;
  - S6 possession corridor availability plan.
- Verifiers for S4, S5, S6, and S7.
- Generated verification reports for S3R, S4, S5, S6, S7, Gate B parity, and Gate A binder validation.
- Focused diffs and Git command summaries.

## Excluded

- Full canonical data.
- Python virtual environment.
- Full repository history.
- UI, Hermes, NL query authoring, or video work.
- Generated caches not needed for inspection.

## Assumptions

- Reviewer is inspecting architecture and evidence quality, not rerunning validation.
- Full reproducibility requires the repository, local Python environment, and canonical Bundesliga-derived data.
