# Scope

## Included

- Backend provenance enforcement for `HERMES_NOVEL_COMPOSITION`.
- Exact attested hero interpretation path backed by committed N1D.1/N1I artifacts.
- Runtime image packaging for N1D proof artifacts.
- Workbench UI state for verified novel composition, unverified experimental drafts, and locked attested scope.
- Required evidence projection fix for possession-start fields.
- Live deployed route smoke over the public Workbench API.

## Excluded

- General-purpose novel-composition attestation for every future Hermes draft.
- Scope-independent structural attestation design.
- New football primitives or tactical families.
- Fresh sealed model evaluation.
- Visual redesign beyond Beta 1A/1B product clarity already represented in copied source.

## Important Interpretation

Beta 1C exposes the exact committed N1I Hermes-origin plan. It does not ask a fresh live model call to reproduce the same AST before every user run. A fresh live call was tested during implementation and produced a valid-looking but structurally different unverified draft; the product gate correctly kept that draft out of `HERMES_NOVEL_COMPOSITION`.
