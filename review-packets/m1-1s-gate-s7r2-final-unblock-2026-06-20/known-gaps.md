# Known Gaps

## requires_full_repo: Command Reproduction

The packet does not contain canonical data, dependency environments, or the full repository. Validation reports are included, but commands cannot be rerun from the packet alone.

Default boundary: inspection-only evidence.

## not_in_scope: M1.2 Hermes

S7R2 unblocks the next milestone; it does not implement Hermes natural-language drafting or revision.

Default boundary: start M1.2 only after approval/owner confirmation.

## not_in_scope: Human Visual Workshop

This packet does not implement the thin replay-backed human inspection surface. That should be part of M1.2 S1, reusing coordinate replay.

Default boundary: M1.2 must include human visual inspection, but S7R2 does not.

## not_in_scope: Polished UI

Final typography, transitions, guided mode, and polish remain later milestones.

Default boundary: M1.2 needs a plain replay panel, not final UI.
