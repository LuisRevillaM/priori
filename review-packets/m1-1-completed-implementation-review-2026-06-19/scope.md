# Scope

## In Scope

This packet asks for review of completed M1.1 only:

- Tactical Query IR v1.
- Binder and catalog validation.
- Runtime execution against the real local corpus.
- M1 parity against the frozen M1 baseline.
- Predicate traces and non-match evaluation.
- `geometric_progressive_corridor`.
- Experimental plan-data composition.
- Static developer inspector.
- Controller verification evidence.

## Out Of Scope

The packet does not ask the reviewer to approve:

- M1.2 Hermes/natural-language workshop implementation.
- Analyst feedback loop implementation.
- Saved user-authored detectors.
- Polished UI/UX.
- D3/canvas/frontend visualization stack decisions.
- Priori integration.
- Match video.
- Cloud deployment, auth, persistence, or production readiness.

## Review Standard

The standard is not "the tests pass." The standard is whether M1.1 gives the next autonomous agents a clear, bounded, low-ambiguity runtime foundation for M1.2 without inviting drift, reward hacking, or another near-term rearchitecture.
