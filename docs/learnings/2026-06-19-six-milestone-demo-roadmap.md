# Learning - Six-Milestone Demo Roadmap

Date: 2026-06-19

## Fact

Planner review recommended replacing the four-milestone roadmap with six milestones because one detector does not prove reusable tactical-query capability and a functional workbench is not the same as a delightful meeting demo.

## Decision

Adopt the six-milestone roadmap:

```text
M1  Verified Ball-Side Block-Shift Evidence Spine
M2  Deterministic Tactical Query Catalog v1
M3  Analyst Workbench v1
M4  Grounded Query Assistant Pilot
M5  Demo Experience, Motion, and Visual QA
M6  Meeting-Ready Independent Demo Release
```

## Follow-Up

- Keep implementation focused on M1 Gate A first.
- Do not begin M2 until M1 review decides whether the proposed second query should remain `Wide-Channel Support Collapse`.
- Treat M4 as a ship/cut gate, not a required feature.
- Reserve M5 for delight and visual QA so final release work does not become hidden feature development.
