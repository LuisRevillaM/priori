# M1.2 S2G - Analyst Vocabulary Boundary

## Learning

The model-backed compiler needs a host-owned vocabulary boundary for the exact
capabilities the system already supports. Without that boundary, the model
over-refuses natural analyst wording such as club-vetted block collapse,
penetrative channel, cover underneath, and useful reach.

The fix should remain bounded. S2G did not create new tactical concepts. It
mapped analyst synonyms to existing recipe families or clarification dimensions,
then used deterministic fallback only when semantic validation had already
proven the correct family.

## Guardrail

Any sealed set that drives vocabulary changes becomes diagnostic. Passing it
after the change is regression evidence, not independent acceptance evidence.
Request another fresh sealed set after the vocabulary boundary freezes.
