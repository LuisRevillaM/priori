# Demo Data Attribution

The Cloud Workbench Alpha data bundle is derived from the public IDSSE/DFL
sample data used throughout this repository.

Runtime bundle contents are limited to the four-match demonstration corpus
needed by the current Workbench recipes:

```text
J03WOY
J03WPY
J03WQQ
J03WR9
```

The bundle includes:

- canonical Parquet tables generated by this project;
- four raw `tracking.xml` files required by the current runtime ball-state
  reader;
- no review packets, Playwright reports, notebooks, secrets, local execution
  artifacts, or full raw provider corpus.

The deployment should retain the upstream dataset terms and the source-lock
metadata already recorded in this project. Do not present the cloud alpha as
using licensed match video; it uses coordinate/event data only.
